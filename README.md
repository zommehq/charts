# Zomme Helm Charts

Helm charts for Zomme projects.

## Available Charts

- **buntime** - Worker Pool Runtime for Bun applications

## Usage

Add this repository to your Helm:

```bash
# Via Rancher UI
# Apps > Repositories > Create
# - Name: zommehq-charts
# - Git Repo: https://github.com/zommehq/charts
# - Branch: main
```
